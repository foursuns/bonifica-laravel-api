
## Enviroment

```bash
$ ren env-example .env
```

## Installation

```bash
$ composer install
```

## Key Generate

```bash
$ php artisan key:generate
```

## Run the Migration

```bash
$ php artisan migrate
```

## Running the app

```bash
$ php artisan serve
```

## Running the Queue

```bash
$ php artisan queue:work --queue=default
```
